🚀 GETTING STARTED IN 5 MINUTES

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: START XAMPP
────────────────────────────────────────────────────────────────
  ✓ Open XAMPP Control Panel
  ✓ Click "Start" for Apache
  ✓ Click "Start" for MySQL
  ✓ Wait for both to show "Running" status

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 2: INSTALL DATABASE (AUTOMATIC)
────────────────────────────────────────────────────────────────
  ✓ Open browser
  ✓ Go to: http://localhost/dormitory-management-system/database/install.php
  ✓ Wait for "Installation Complete" message
  ✓ Note the default credentials displayed
  ✓ DELETE the install.php file after completion

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 3: LOGIN
────────────────────────────────────────────────────────────────
  ✓ Go to: http://localhost/dormitory-management-system/login.php
  
  ADMIN CREDENTIALS:
  • Email: admin@dormitory.com
  • Password: Admin123!
  
  TENANT CREDENTIALS (for testing):

STEP 4: EXPLORE THE SYSTEM
────────────────────────────────────────────────────────────────
  AS ADMIN:
  ✓ View Dashboard → See system statistics
  ✓ Manage Rooms → Add new rooms
  ✓ Manage Tenants → View tenant list
  ✓ Process Bookings → Approve/reject requests
  ✓ Track Payments → Monitor payment records
  ✓ Post Announcements → Communicate with tenants
  ✓ Generate Reports → View analytics
  
  AS TENANT:
  ✓ Browse Rooms → View available rooms
  ✓ Book a Room → Submit booking request
  ✓ View Portal → Check booking status
  ✓ Update Profile → Manage personal info

⚠️  TROUBLESHOOTING
────────────────────────────────────────────────────────────────
  Problem: "Database connection failed"
  Solution: Check MySQL is running, verify config/database.php

  Problem: "Page not found"
  Solution: Ensure Apache is running, check project location

  Problem: White screen
  Solution: Check Apache error logs, enable PHP error display
